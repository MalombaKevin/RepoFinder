Repofinder

Repofinder is a github specific search engine that enables users to search for user's profiles and repositories on Github.
Github is an online software development platform used for storing, tracking, and collaborating on software projects.
It enables developers to upload their own code files and to collaborate with fellow developers on open-source projects.

How Repofinder Works

[A] On the Navigation Bar, you will see two options centered (User and Repo)



[B] When you select User, you can search for a Github User using
their Github username
[C] When you select Repo, you can search for GitHub repositories from Github user's profiles
